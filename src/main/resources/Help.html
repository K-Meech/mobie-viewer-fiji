Actions

PlatyBrowser keyboard commands (working in addition to default BigDataViewer commands)

P Print current position in micrometer units
D Discover genes expressed at current position
Ctrl + Left-click Toggle selection
Crtl + Q >Unselect all
Ctrl + S Iterate selection mode:
  • Dim not selected
  • Only show selected
  • Color selected bright yellow
  • Color selected bright yellow and dim not selected
Ctrl + Shift + S Shuffle colors of random LUT
Ctrl + Shift + Left-click View object in 3D
